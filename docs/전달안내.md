# Portly 설치 안내 (타팀 전달용)

> 아래 내용을 그대로 복사해서 `.dmg` 파일과 함께 전달하세요.

---

## 📦 Portly 설치법

로컬 dev 서버를 메뉴바에서 보고 끄는 앱이에요. macOS 전용(Intel·Apple Silicon 모두 지원).

### 설치

1. 받은 **`Portly_0.1.0_universal.dmg`** 를 더블클릭
2. 열린 창에서 **Portly 아이콘을 Applications 폴더로 드래그**
3. `런치패드` 또는 `응용 프로그램`에서 Portly 실행

### ⚠️ 첫 실행 시 "확인되지 않은 개발자" 경고가 뜨면

이 앱은 Apple 서명이 안 되어 있어서 macOS가 처음에 막아요. 아래처럼 열면 됩니다:

**방법 1 (대부분)**
- 응용 프로그램 폴더에서 **Portly을 우클릭 → "열기"** → 경고창에서 다시 **"열기"**
- (한 번만 하면 다음부터는 그냥 열려요)

**방법 2 (macOS Sequoia 이상에서 방법 1이 안 되면)**
- `시스템 설정` → `개인정보 보호 및 보안` → 아래로 스크롤
- "Portly이(가) 차단되었습니다" 옆 **"확인 없이 열기"** 클릭

### 쓰는 법

- 메뉴바(상단 오른쪽)에 아이콘이 뜹니다 → **클릭하면 포트 목록**
- **아이콘이 안 보이면**: 메뉴바가 꽉 차서 그래요(특히 노치 맥북). 다른 메뉴바 앱을 정리하거나, 화면 녹화/라이브 액티비티를 끄면 나타나요.
- 내 dev 서버가 위에, 시스템/기타는 접혀 있어요. **"종료"** 버튼으로 끄면 됩니다.

문제 있으면 (전달자)에게 문의하세요.
